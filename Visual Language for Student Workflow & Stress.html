Visual Language for Student Workflow & Stress

Iconic Symbols

Timeline

1 week (Oct 2024)

Technologies

Adobe Illustrator

Project Overview

This project explores the emotional and psychological impact of student workflow and stress through a symbolic icon set.

Design Process


The icon set uses Color Psychology to convey different emotional states:

Blue Colour Scheme

Symbolizes productivity and focus

Red Colour Scheme (with Edited and Additional Icon)

Represents stress and anxiety.

Brown Colour Scheme

Serves as a neutral color for balance and calmness.

Weekly lesson topics(Extra)

Created as part of a Computation Art course project exploring weekly lesson topics through icon design.
